Hydro Cli

在使用 cli 之前，请完成数据库配置。
指令中使用 <> 括起来的参数必须给出，用 [] 括起来的参数可以给出，若不给出则按照默认设置。
用户请根据自己的情况替换掉用 <> 或是 [] 包括起来的部分（括号也应替换）
